# Student Manager App

## Project Description

Student Manager App is a simple Python console application that helps manage student records. Users can add student names and marks, view all student records, and check whether a student has passed or failed based on their marks. This project demonstrates the use of Python dictionaries, loops, conditional statements, and user input handling.

## Features

* Add a new student with marks
* View all student records
* Check a student's result (Pass/Fail)
* Menu-driven interface
* Continuous execution until the user chooses to exit

## Technologies Used

* Python 3

## How to Run

1. Download or clone the project.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

```bash
python student_manager.py
```

5. Follow the on-screen menu options.
